HTML과 CSS



HTML과 CSS는 마크업 언어로, 정확하게는
프로그래밍 언어가 아닙니다
태그를 사용하며 주로 웹 사이트를 구성 할 때 사용되는 언어로
HTML은 글, 표, 사진 등 기초적인 틀을 잡는 언어고
CSS는 폰트, 색, 글자 크기 등 꾸미는 언어입니다
문법이 정말 간단해서 입문용으로 많이 추전되는 언어입니다
실제로 제가 가장 처음 배운 언어이기도 합니다
연습용으로 학교 시간표를 만들어 봤었고
지금 이 웹 사이트도 HTMl과 CSS로 구성되어 있습니다

Python



Python(파이썬)은 현재 메이저 프로그래밍 언어들 중에서
독보적으로 간편하고 쉬운 문법을 가졌으며
각종 여러 프로그램을 만드는데 사용되는 언어입니다
그렇기 때문에 HTML다음으로 배운 언어입니다
텍스트 기반 어드벤처 게임과
수수께끼 풀기 게임 등을 만들어 봤습니다

C#



C#은 유니티(Unity)에서 사용하는 언어로
유니티를 통해 게임을 개발하려면 필수로
배워야 하는 언어입니다
앞의 두 언어보다 문법이 어렵고 복잡하며
난이도가 꽤 되는 언어입니다
현재 제가 배우고 있는 언어이기도 합니다

(앞의 두 언어보다 정말 굉장히 어렵기 때문에 고생하고 있습니다)
